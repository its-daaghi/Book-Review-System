Introduction <br>
This project is a Django-based Book Review System developed to understand and implement core CRUD (Create, Read, Update, Delete) operations in a real-world web application. It focuses on clean structure, correct use of Django fundamentals, and practical database interactions.
This repository represents a learning-focused project, built with attention to best practices and clarity rather than complexity.

# Purpose of the Project
The primary goal of this project is to:<br>
Gain hands-on experience with Django<br>
Understand CRUD operations using Django ORM<br>
Learn how models, views, templates, and URLs work together<br>
Build confidence in backend development fundamentals<br>

# Key Features
CRUD Operations<br>
Create: Add new books with essential details<br>
Read: View a list of books and detailed book information<br>
Update: Edit existing book records<br>
Delete: Remove books from the system<br>

# Additional Highlights
Structured Django project layout<br>
Clean separation of logic (models, views, templates)<br>
Django Admin integration for data management<br>

# Technology Stack
Framework: Django (Python)<br>
Database: SQLite<br>
Frontend: HTML, CSS (Django Templates)<br>
ORM: Django ORM<br>

# What This Project Demonstrates
Understanding of Django fundamentals<br>
Proper implementation of CRUD functionality<br>
Ability to structure and document a project professionally<br>
Willingness to learn, practice, and improve<br>

# Limitations
This project is intentionally kept simple and focuses on backend fundamentals. Advanced features such as authentication, authorization, and APIs are not included in this version.
